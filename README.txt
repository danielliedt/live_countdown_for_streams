Live Countdown Timer
====================

A clean, responsive countdown timer with elegant styling and glowing effects.

Installation
------------
1. Download the HTML file
2. Add source 'Browser' on OBS
3. 'Browser': Properties - ‚òê local file - add 'countdown.html'

Customization
-------------
To change the target date, edit line 88 in the JavaScript:
const targetDate = new Date("2025-12-27T18:00:00");

You can also customize:
- Colors by modifying the CSS color values
- Fonts by changing the font-family
- Styling by adjusting the CSS properties
- Add background picture

Browser Support
---------------
Works in all modern browsers including:
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

Technical Details
-----------------
- Pure HTML, CSS, and JavaScript (no frameworks)
- CSS Grid and Flexbox for layout
- CSS clamp() for responsive typography
- CSS animations for glow effects
